# TodoMVC React

This is an implementation of [TodoMVC](https://todomvc.com/) built with:

- XState v5
- React
- TypeScript
- Vite

## How to run

You may run the example locally with:

```shell
    cd examples/todomvc-react
    pnpm install
    pnpm run dev
```
