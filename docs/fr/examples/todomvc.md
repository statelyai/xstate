# Examples de TodoMVC

Les deux démos partagent les mêmes machines.

## TodoMVC avec React

- Utilisation des [hooks](https://reactjs.org/hooks) react.
- Deux machines:
  - `todosMachine` contrôle l'ensemble de l'application Todos
  - `todoMachine` contrôle chaque élément Todo.
